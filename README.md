2. 安装方式
  * Linux && OSX
```sh
cd ~
git clone https://github.com/leebo/bovim.git
cd bovim && ./init.sh
```
  * windows
```sh
cd %HOMEPATH%
git clone https://github.com/leebo/bovim.git
cd bovim && ./init.bat
```
